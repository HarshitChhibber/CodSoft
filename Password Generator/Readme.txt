# Password Generator (CLI)

A simple, command-line-based Password Generator built using Python. This tool helps you create strong and customizable passwords and optionally allows you to save them for future reference.

## Features

- Set custom password length.
- Choose which character types to include:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Digits (0-9)
  - Special characters (!, @, #, etc.)
- Ensures at least one character from each selected type is present.
- Option to save the generated password to a `.txt` file along with the associated website name.

## Technologies Used

- Python 3.11
- Standard libraries: `random`, `string`
